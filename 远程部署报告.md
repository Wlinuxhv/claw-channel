### 1. 安全建议

- ⚠️ 配置防火墙（仅允许信任 IP）
- ⚠️ 启用 HTTPS（生产环境）

### 2. 数据备份

建议定期备份数据库：

```bash
# 添加到 crontab
0 2 * * * cp /home/wlinuxhv/claw-channel/data/claw.db /backup/claw-$(date +\%Y\%m\%d).db
```

### 3. 日志管理

日志文件会持续增长，建议定期清理：

```bash
# 保留最近 30 天日志
find /home/wlinuxhv/claw-channel/logs -name "*.log" -mtime +30 -delete
```

### 4. 自动启动

配置 systemd 服务实现开机自启：

```bash
# 创建服务文件
sudo tee /etc/systemd/system/claw-channel.service > /dev/null <<'EOF'
[Unit]
Description=Claw Channel Server
After=network.target

[Service]
Type=simple
User=wlinuxhv
WorkingDirectory=/home/wlinuxhv/claw-channel
ExecStart=/home/wlinuxhv/claw-channel/claw-server
Restart=on-failure
RestartSec=10

[Install]
WantedBy=multi-user.target
EOF

# 启用服务
sudo systemctl daemon-reload
sudo systemctl enable claw-channel
sudo systemctl start claw-channel
```

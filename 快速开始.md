# 快速开始指南

## 🚀 5 分钟快速体验

### 步骤 1: 启动服务端

```bash
cd "/app/working/Claw Channel/Server"

# 方式一：直接运行（需要 Go 环境）
go run cmd/server/main.go

# 方式二：使用 Docker（推荐）
cd ..
docker-compose up -d
```

### 步骤 2: 测试 API

```bash
cd "/app/working/Claw Channel/Server"
./test.sh
```

### 步骤 3: 测试 WebSocket

安装 wscat：
```bash
npm install -g ws
```

连接 WebSocket：
```bash
wscat -c ws://localhost:8080/ws/chat
```

发送消息：
```json
{
  "type": "message",
  "content": "你好，龙虾！",
  "content_type": "text"
}
```

## 🔧 配置说明

### 服务端配置

编辑 `Server/internal/config/config.go`：

```go
// 修改用户凭证（生产环境）
cfg.AllowedUsers["claw_user_001"] = "your_secure_secret_001"
```

### 环境变量

```bash
export SERVER_PORT=8080
export JWT_SECRET=your-secret-key
export OPENCLAW_WS_URL=ws://localhost:18789/ws
```

## ❓ 常见问题

### Q: 服务端启动失败？

A: 检查端口是否被占用：
```bash
lsof -i :8080
```

### Q: 无法连接 OpenClaw？

A: 确保 OpenClaw 已启动：
```bash
docker ps | grep openclaw
```

### Q: WebSocket 连接失败？

A: 检查 Token 是否有效，查看服务端日志
